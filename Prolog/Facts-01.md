Facts in Prolog: 

parent (laura, damian). -> Laura is Damian's parent
"parent" is the relation name (predicate name)

- "laura" and "damian" are the arguments
- facts in prolog finsih with a period 

- Name of relations and arguments that refer to objects begin with lowercase 

- Set of facts or caluses, together with a set of rules make up a PROLOG database 

is_in_love_with(romeo, juliet).
// romeo is in love with juliet 
// 1st argument is in love with the 2nd argument

is_valuable(gold).
// gold is valuable 

has(john, book).
// john has a book
// 1st argument has 2nd argument 

gives(john,book,sara).
// john gives a book to sara 
// 1st argument takes the 2nd argument and gives it to the 3rd argument 

Realtion and facts are not the same. Relation is the "function name"
Facts are what we provide as arguments
